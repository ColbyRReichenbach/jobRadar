- [CUR-AUGUSTA-STATIC-ANALYTICS] Built a Vite/React data-storytelling site with home, profile, repeat-context pages, hole-by-hole scorecards, player/field/top-10 comparisons, repeat champion benchmarks, and static visual assets.
  evidence_skills: Vite, React, TypeScript, data storytelling, static site
- [CUR-AUGUSTA-JSON-DATA-LAYER] Built a static JSON data layer imported at build time for tournament, winner, scorecard, profile, all-player scorecards, tournament stats, course hole stats, methodology, repeat champion benchmarks, recent champions, annotations, repeat context, and comparison data.
  evidence_skills: JSON data modeling, static data layer, data validation
- [CUR-AUGUSTA-SELECTORS] Implemented selectors for round/hole selection, hole difficulty, hole comparisons, trajectory points, gain/loss series, cohort players, metric distributions, weighted averages, winner profile metrics, and repeat comparison rows.
  evidence_skills: selector functions, derived metrics, analytics logic, TypeScript
- [CUR-AUGUSTA-PLAYWRIGHT-PIPELINE] Built a Playwright/Node pipeline that fetches an ESPN leaderboard, parses player scorecard URLs, fetches player scorecards with concurrency/throttle controls, fetches Masters hole stats, validates metadata, builds artifacts, writes outputs, and optionally syncs them into app/public data.
  evidence_skills: Playwright, Node ESM, data ingestion, concurrency controls, artifact generation
- [CUR-AUGUSTA-VALIDATION-BOUNDARY] Documented unsupported values such as hole-level fairway hit, GIR, putts, field fairway percentage, field GIR percentage, top-10 fairway percentage, top-10 GIR percentage, and automatic editorial narrative because public pages did not provide clean trustworthy sources.
  evidence_skills: source-boundary documentation, data validation, evidence governance
- [CUR-AUGUSTA-AGGREGATION] Built deterministic aggregation for hole averages, metric averages, scoring distributions, trajectories, and winner verdict text.
  evidence_skills: deterministic aggregation, metric calculations, static analytics
- [CUR-AUGUSTA-VISUALIZATIONS] Built visualizations for cumulative tournament path, hole gain/loss, round swing story, hole delta matrix, metric distributions, player comparison dumbbell, repeat path, repeat champion benchmark, and scorecards.
  evidence_skills: Recharts, data visualization, analytics UI
- [CUR-AUGUSTA-SOURCE-BOUNDARY] Preserved the boundary that generated data artifacts are reported as codebase artifacts and were not externally revalidated in the report.
  evidence_skills: evidence governance, source-boundary documentation, validation policy
